# Punjab Probation & Parole Service Website

A premium, highly interactive one-pager website for the **Punjab Probation & Parole Service (PP&PS)** under the Home Department, Government of the Punjab.

## Features

- **Interactive Citizens & Legal Portal**:
  - **Case Eligibility Estimator**: A dynamic evaluation tool matching legal parameters (offense level, conduct, time served) to output probation or parole feasibility ratings.
  - **Case Status Progress Tracker**: A visual status timeline search matching mock cases (e.g., `PPPS-2026-0042`, `PPPS-2026-0118`, `PPPS-2026-0789`) to update rehabilitation steps.
- **Modernized Digital Reforms Panel**: Showcases the Offenders Management Information System (OMIS), E-Office implementation, and the PMS Field Officer App.
- **Historical Timeline**: Covers the evolution of parole and probation enactments in Punjab (1926, 1960, and 2019).
- **Responsive Dark & Light Themes**: Accessible styling options with glassmorphism overlays and vibrant accents.
- **SEO Ready Structure**: Semantic HTML5 markup, localized meta tags, and structured headings.

## Tech Stack

- **Core**: HTML5, Vanilla JavaScript (ES6)
- **Styling**: CSS3 (Variables, Flexbox, CSS Grid, Custom Transitions, Backdrop-filter blurs)
- **Assets**: Inline custom SVGs for maximum load performance and styling control.

## Running Locally

Simply open the `index.html` file in any modern web browser to view the application:
```bash
# Double-click index.html or open via terminal
start index.html
```
